# Project Overview

## Name

DevAtlas

## Tagline

The map for your Guidewire frontend codebase.

## Vision

DevAtlas provides developers with instant context around any Jutro file, component, page, or feature.

Instead of manually navigating:

Code → Git → Jira → Confluence → Senior Engineer

Developers receive all context in a single interface.

## Core Value Proposition

Reduce investigation time from hours to minutes.

## Target Users

Primary:

* Frontend Engineers

Secondary:

* Tech Leads
* Architects
* New Joiners

## Key Benefits

* Faster onboarding
* Reduced context switching
* Better impact analysis
* Less dependency on tribal knowledge
* Improved confidence before code changes
