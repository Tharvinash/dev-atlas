# Development Phases

## Phase 1 - UI Foundation

Goal:
Build clickable hackathon UI.

Deliverables:

* Layout
* Sidebar
* Code panel
* Context panel
* Mock data

Success:
User can click a file and view analysis.

---

## Phase 2 - Repository Analysis

Goal:
Read local Jutro repository.

Deliverables:

* File scanning
* File indexing
* Component discovery
* Import analysis

Success:
Repository contents populate UI.

---

## Phase 3 - Git Integration

Goal:
Add source control intelligence.

Deliverables:

* git log integration
* git blame integration
* commit parsing

Success:
File shows recent commits and ownership.

---

## Phase 4 - Jira Integration

Goal:
Correlate commits with tickets.

Deliverables:

* Extract ticket IDs
* Query Jira
* Display ticket metadata

Success:
File displays related Jira tickets.

---

## Phase 5 - Confluence Integration

Goal:
Correlate files with documentation.

Deliverables:

* Search Confluence
* Match pages
* Display links

Success:
File displays related documentation.

## Phase 7 - Demo Polish

Hackathon-ready experience.
