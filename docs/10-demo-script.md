# Demo Script

## Opening

Today engineers spend significant time acting as detectives.

To understand a single file they must navigate:

* Code
* Git
* Jira
* Confluence

and often ask senior engineers for missing context.

## Problem

Receive ticket:

GW-1245

Update Claim Payment Display.

Question:

Where should I make changes?

What else could break?

## Solution

Open DevAtlas.

Search:

ClaimSummaryPage.tsx

Instantly receive:

* Business context
* Git history
* Jira links
* Confluence documentation
* Dependency graph
* Impact analysis

## Outcome

Investigation time reduced from hours to minutes.

Knowledge becomes discoverable instead of tribal.
